Check repo change 配置文件添加要检查的软件源

添加到 crc 的配置文件中

1.从 Filza 中打开文件

<bark_data>/Library/Preferences/com.nan.crc.plist

<bark_data> 即 Bark 存放数据的文件夹

2.找到 source，然后点击右侧的 i

3.点击 Add item

4.点开新出现的 item，点击类型，修改为 Dictionary

5.再次点击下方的 Add item

6.点开新出现的 item，将名称设置为 enable，类型设置为 Boolean，下方值设置为 YES

7.接下来返回上一页，并点击 Add item

8.点开新出现的 item，将名称设置为 url，类型不用变，下面的值写入软件源的 url

9.保存配置文件

返回首页
