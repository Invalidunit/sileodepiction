包过滤（architecture）

用于过滤无根包或者有根包，防止不属于自己环境的包推送打扰自己。

可设置为：

auto：自动匹配当前越狱环境

iphoneos-arm：过滤为有根越狱包

iphoneos-arm64：过滤为无根越狱包

iphoneos-arm64e：过滤为隐根越狱包（RootHide）

all：禁用过滤，将输出所有包

默认情况下为 auto，将根据当前越狱的环境来进行匹配，防止其他类型的包更新后知道他更新了但是用不了的难受

更改请在配置文件中找到 filter_architecture，并设置为对应值

包名通知

可以在通知时尝试使用缓存中的包名，而不是包 id 进行通知

可能对某些小白比较友好吧

但是因为使用的方法更偏向速度，所以有些源上架了不包含包名的包时，会直接打乱我整张缓存表

所以加上了检测，如果发现这种情况则这个源的缓存不可信，继续使用包 id 进行通知

启用请在配置文件中找到 use_name，并设置为 YES
