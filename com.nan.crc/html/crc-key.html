一、从 App Store 安装 Bark

1.请先从 App Store 中安装 Bark App，这是为你推送通知的方式。

打开 Bark 安装页

点击上方跳转 App Store

二、获取 Bark 推送 key

1.打开 Bark app 后点击中心的圆形注册按钮

2.按下图黄色框框圈起来的位置复制链接

3.将复制出来的链接仅保留上图蓝色框框部分（两个斜杠中间夹着的部分就是 key，不包含斜杠）

三、将 key 写入 crc 配置文件

1.从 Filza 中打开文件

<bark_data>/Library/Preferences/com.nan.crc.plist

<bark_data> 即 Bark 存放数据的文件夹

2.点击 bark 展开子菜单，找到 key，然后点开，找到 item 0 也就是下图蓝框位置

3.点击 item 0 右侧的 i 也就是上图黄框位置，并粘贴第二步中获得的 key

4.配置完 key 后可以重装软件包，将会触发一条测试用的通知，告诉你开始工作了
