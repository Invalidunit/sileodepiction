Check AppStore change 配置文件添加要检查的软件

一、获取软件分享链接

1.请先从 App Store 中找到想要检查的软件并打开软件介绍界面

2.点击右上角的分享按钮，并点击“拷贝链接”

二、添加到 cac 的配置文件中

1.从 Filza 中打开文件

<bark_data>/Library/Preferences/com.nan.cac.plist

<bark_data> 即 Bark 存放数据的文件夹

2.找到 app，然后点击右侧的 i

3.点击 Add item

4.点开新出现的 item，点击类型，修改为 Dictionary

5.再次点击下方的 Add item

6.点开新出现的 item，将名称设置为 enable，类型设置为 Boolean，下方值设置为 YES

7.接下来返回上一页，并点击 Add item

8.点开新出现的 item，将名称设置为 url，类型不用变，下面的值粘贴刚刚从 AppStore 拷贝的链接

9.保存配置文件
